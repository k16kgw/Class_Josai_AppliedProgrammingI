# 第13回　休講：レポート作成

第13回は休講とし，各自で最終レポートを作成する時間に充てる．

最終レポートでは，各自でテーマを設定し，データ取得，前処理，集計，可視化，考察までを一気通貫で行う．
第10回から第12回で扱った行政統計，経済データ，スポーツデータの実践例を参考にしてよい．

## レポートで含める内容

| 項目 | 書く内容 |
| --- | --- |
| テーマ | 何を明らかにしたいか |
| データ | 使用したデータの名称，出典，URL，取得日 |
| 前処理 | どの列を使い，どのように整形したか |
| 可視化 | 作成した図，図を選んだ理由 |
| 考察 | 図から読み取れる傾向 |
| 限界 | この分析だけでは判断できないこと |
| 再現性 | 作成したPythonファイル，保存した図表，フォルダ構成 |

## 提出物

提出物の形式は授業内で指定する．
少なくとも，次の内容を含めることを想定する．

- レポート本文
- 使用したPythonファイル
- 作成した図表
- 使用データの出典情報

## 注意

- 出典が不明なデータは使わない
- ライセンスや利用条件を確認する
- 図だけでなく，図から読み取れることを文章で説明する
- 分析結果を過度に一般化しない
- 自分の分析で言えることと言えないことを分けて書く
