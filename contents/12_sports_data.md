# 第12回　スポーツデータの分析実践

### 今回の位置づけ

第12回では，スポーツデータを使って分析の流れを学ぶ．
スポーツデータは，試合結果，チーム成績，選手成績，順位表など，集計や比較に向いたデータが多い．

この回では，**テーマ設定，データ取得，前処理，可視化，考察**までを扱い，最終レポートで必要になる「図表と文章で説明する」力を確認する．

### 到達目標

- スポーツデータを使った分析テーマを設定できる
- チーム名，選手名，試合日，得点などの列を適切に扱える
- グループ別集計やランキングを作成できる
- 棒グラフ，ヒストグラム，散布図を使って成績の特徴を可視化できる
- 分析結果を，図表と文章を組み合わせて説明できる

---

## テーマ設定

スポーツデータでは，次のような問いを立てることができる．

| 観点 | 問いの例 |
| --- | --- |
| チーム比較 | 得点が多いチームと少ないチームにはどのような違いがあるか |
| 選手比較 | 選手ごとの成績にどのような分布があるか |
| 試合結果 | 勝敗と得点差にはどのような関係があるか |
| 時間変化 | シーズン中に成績はどのように変化したか |

スポーツデータでは，ランキングを作るだけでなく，分布や関係を見ることが重要である．
上位だけに注目すると，全体の傾向を見落とすことがある．

---

## 演習の流れ

この回では，授業で指定するスポーツデータを使い，次の流れで分析する．

```text
テーマ設定
  ↓
データ取得
  ↓
pandasで読み込み
  ↓
列名・欠損値・データ型の確認
  ↓
チーム別・選手別の集計
  ↓
ランキングと分布の確認
  ↓
棒グラフ・ヒストグラム・散布図で可視化
  ↓
読み取れることを文章で整理
```

### 演習1：データを読み込む

```python
import pandas as pd

df = pd.read_csv("data/raw/sports_data.csv")

print(df.shape)
print(df.columns)
df.head()
```

### 演習2：基本的な確認を行う

```python
df.info()
df.describe()
```

カテゴリ列についても確認する．

```python
df["チーム名"].value_counts()
```

### 演習3：チーム別に集計する

```python
team_summary = (
    df.groupby("チーム名", as_index=False)
    .agg(
        試合数=("試合ID", "count"),
        平均得点=("得点", "mean"),
        合計得点=("得点", "sum"),
    )
)

team_summary.head()
```

### 演習4：ランキングを可視化する

```python
import seaborn as sns

top_teams = team_summary.sort_values("合計得点", ascending=False).head(10)

sns.barplot(data=top_teams, x="合計得点", y="チーム名")
```

### 演習5：分布と関係を可視化する

```python
sns.histplot(data=df, x="得点")
```

```python
sns.scatterplot(data=team_summary, x="試合数", y="平均得点")
```

### 演習6：考察を書く

図から読み取れることを，次の形で整理する．

1. 上位のチームや選手にはどのような特徴があるか
2. 成績の分布に偏りはあるか
3. 2つの指標の間に関係は見られるか
4. このデータだけでは判断できないことは何か
