### メモ

- JPXサイト掲載情報の権利，商用目的でのデータ収集・二次利用，免責事項，高頻度・高負荷な自動取得に関する注意を確認する必要がある．
- 個別銘柄の株価，チャート，株価指数，先物価格などは，JPX以外の情報提供元から提供されているものが含まれる．そのため，データの種類によっては「提供情報の注意事項及びデータ更新タイミングについて」も確認する．
- 授業・レポートで使う場合は，少なくとも次を記録する．

| 記録する項目 | 内容 |
| --- | --- |
| データ名 | 例：統計月報，月間相場表，投資部門別売買状況など |
| 提供者 | 日本取引所グループ，東京証券取引所など |
| 取得元URL | 使用した統計ページまたはファイルのURL |
| 取得日 | 自分がデータを取得した日 |
| 利用条件確認ページ | 上記の注意事項・免責事項ページ |
| 使用目的 | 授業内レポート，非商用の分析など |

### 学生に案内する場合の注意

JPX統計情報は金融市場データとして有用だが，商用利用，二次利用，第三者提供，自動取得，指数データの権利関係に注意が必要である．
学生が最終レポートで使う場合は，利用条件を確認したうえで，出典と取得日を明記させる．
