# 第11回　経済データの分析実践

### 今回の位置づけ

第11回では，経済データを使って分析の流れを学ぶ．
経済データには，物価，賃金，為替，株価，消費，景気など，時間に沿って変化するデータが多い．

この回では，時系列データを中心に，**テーマ設定，データ取得，前処理，可視化，考察**までを扱う．

### 到達目標

- 経済データを使った分析テーマを設定できる
- 日付や期間を表す列を適切に扱える
- 時系列データの欠損値や単位を確認できる
- 折れ線グラフや散布図を使って変化や関係を可視化できる
- 経済データから読み取れる傾向と，読み取りに注意が必要な点を説明できる

---

## テーマ設定

経済データでは，次のような問いを立てることができる．

| 観点 | 問いの例 |
| --- | --- |
| 時間変化 | 物価や消費は時間とともにどのように変化しているか |
| 比較 | 複数の地域や品目で変化の仕方は異なるか |
| 関係 | 2つの指標には似た動きがあるか |
| 変化点 | ある時期を境に傾向が変わっているか |

経済データでは，単位や基準年が重要である．
同じ「指数」でも，基準年が異なるとそのまま比較できない場合がある．

### 演習2：pandasで読み込む

```python
import pandas as pd

df = pd.read_csv("data/raw/economic_data.csv")

print(df.shape)
print(df.columns)
df.head()
```

### 演習3：日付列を変換する

時系列データでは，日付や年月を文字列のまま扱わず，日付型に変換する．

```python
df["日付"] = pd.to_datetime(df["日付"])

df = df.sort_values("日付")

df.head()
```

### 演習4：折れ線グラフで変化を見る

```python
import seaborn as sns

sns.lineplot(data=df, x="日付", y="値")
```

### 演習5：複数の指標を比較する

指標名や地域名の列がある場合は，色でグループを分けて比較する．

```python
sns.lineplot(data=df, x="日付", y="値", hue="指標名")
```

### 演習6：考察を書く

図から読み取れることを，次の形で整理する．

1. 全体として上昇・下降・横ばいのどれに近いか
2. 大きく変化している時期はあるか
3. 複数の指標に似た動きはあるか
4. このデータだけでは判断できないことは何か

## まとめ

- 経済データでは，日付，単位，基準年，対象期間を確認することが重要である
- 時系列データでは，日付列を変換し，時間順に並べてから可視化する
- 折れ線グラフは，時間に沿った変化を見るときに有効である
- 複数の指標を比較するときは，単位や基準年の違いに注意する
