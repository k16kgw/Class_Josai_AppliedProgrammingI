### 今回の位置づけ

第10回から第12回では，気象以外のデータを使って，データ分析の実践例を学ぶ．
各回では，**テーマ設定，データ取得，前処理，可視化，考察**までを一つの流れとして扱う．

第10回では，行政統計データを扱う．
行政統計は，人口，世帯，産業，教育，福祉，交通など，社会の状態を把握するために作成されているデータである．

### 到達目標

- 行政統計データを使った分析テーマを設定できる
- データの出典，調査対象，単位，時点を確認できる
- 市区町村コードや都道府県名など，地域を表す列を扱える
- 数値列を前処理し，地域別・項目別に集計できる
- `pandas`，`seaborn`，`plotly`を使って行政統計を可視化できる

---

## テーマ設定

行政統計では，次のような問いを立てることができる．

| 分野 | 問いの例 |
| --- | --- |
| 人口 | 人口が多い地域と少ない地域にはどのような違いがあるか |
| 世帯 | 単身世帯の割合は地域によってどのように異なるか |
| 産業 | 地域ごとに産業構成はどのように異なるか |
| 福祉 | 高齢化率と福祉関連指標にはどのような関係があるか |

問いを立てるときは，最初から大きすぎるテーマにしない．
まずは「地域別に比較する」「時点別に比較する」「2つの指標の関係を見る」のように，図にできる問いへ落とし込む．

### 演習2：pandasで読み込む

```python
import pandas as pd

df = pd.read_csv("data/raw/administrative_statistics.csv")

print(df.shape)
print(df.columns)
df.head()
```

### 演習3：前処理を行う

必要な列を選び，数値として扱う列を変換する．

```python
cols = ["地域名", "地域コード", "指標名", "値"]
df_selected = df[cols].copy()

df_selected["値"] = pd.to_numeric(df_selected["値"], errors="coerce")

df_selected.head()
```

### 演習4：可視化する

地域別の値の分布や，上位の地域を可視化する．

```python
import seaborn as sns

sns.histplot(data=df_selected, x="値")
```

```python
top_df = df_selected.sort_values("値", ascending=False).head(10)

sns.barplot(data=top_df, x="値", y="地域名")
```

### 演習5：考察を書く

図から読み取れることを，次の形で整理する．

1. どの地域が大きい値を示しているか
2. 値の分布に偏りはあるか
